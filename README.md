## Getting Started

  ### Installing
  명령어를 사용해 dependencies를 설치한다.
  
  ```
  npm install
  ```
  
  ### playing
  명령어를 사용해 각 파일별로 실행시킬 수 있다.

  ```
  npm start
  ```
